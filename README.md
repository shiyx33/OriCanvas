# OriCanvas

基于 Web 的折纸折痕图案编辑器，由 [Oriedita](https://github.com/oriedita/oriedita) 移植而来。

> 本项目借助 Claude Code + DeepSeek V4 Pro 完成移植，目前仍在开发中。

纯 HTML/CSS/JS 构建，零依赖，直接在浏览器中运行。

## 功能

- **7 种绘制工具** — 折线、兔耳折、平折线、延长线、垂线、角平分线、平行线
- **4 种选择/橡皮擦模式** — 点选、框选、套索、画笔
- **变换工具** — 移动、复制、镜像
- **折叠模拟引擎** — 右手法则面片提取、几何折叠、层级排序、SubFace 三角剖分
- **Maya 风格二级饼菜单** — Ctrl+右键 / 空格键呼出
- **导入导出** — `.ori` / `.cp` / `.fold` 格式
- **中英文切换**、暗色模式、触屏手势

## 快捷键

| 按键 | 功能 |
|------|------|
| Ctrl+右键 | 饼菜单 |
| 空格 | 饼菜单 |
| F | 折叠模拟 |
| Q | 清理杂点 |
| 中键拖拽 | 画布平移 |
| 滚轮 | 缩放 |

## 技术栈

Canvas 2D 渲染，`lineCap='round'`，支持 `devicePixelRatio` 高清。
